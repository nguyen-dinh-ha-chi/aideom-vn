# AIDEOM-VN — Mô hình Ra Quyết Định Phát Triển Kinh Tế Việt Nam

Bài tập cuối kỳ môn Các Mô Hình Ra Quyết Định  
Trường Đại học Kinh tế — Viện Quản trị Kinh doanh

## Cấu trúc dự án
- `Data/` — Dữ liệu thực tế Việt Nam 2020-2025
- `Output/` — Biểu đồ kết quả
- `pages/` — Các trang dashboard
- `Bài cuối kỳ.py` — Notebook 12 bài tập
- `dashboard.py` — Module dashboard
- `Trang_chu.py` — Giao diện chính Streamlit

## Cài đặt
pip install -r requirements.txt

## Chạy dashboard
streamlit run Trang_chu.py
